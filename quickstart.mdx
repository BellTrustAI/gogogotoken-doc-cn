GogoToken 是一个兼容 OpenAI 接口标准的 AI API 中转平台。只需修改一行代码，即可通过统一端点访问 Claude 和 GPT 系列模型，按量计费，无需订阅。

<Steps>
  <Step title="注册账号">
    前往 [GogoToken 注册页面](https://gogotoken.org/register) 申请账号。目前平台处于**邀请制内测阶段**，请确保您已获得邀请码后再进行注册。
  </Step>

  <Step title="充值余额">
    注册并登录后，前往 [账户余额页面](/account/balance) 为账户充值。GogoToken 采用先充值后使用的按量计费模式，无最低消费要求。
  </Step>

  <Step title="创建 API 密钥">
    前往 [API 密钥管理页面](/account/api-keys) 创建一个新的 API 密钥。请妥善保存您的密钥，它只会在创建时完整显示一次。
  </Step>

  <Step title="发送第一个请求">
    将 OpenAI SDK 的 `baseURL` 替换为 GogoToken 的端点地址 `https://gogotoken.org/v1`，即可立即开始使用。以下示例展示了如何调用 `claude-opus-4-8` 模型：

    <CodeGroup>
      ```javascript JavaScript / Node.js
      import OpenAI from 'openai';

      const client = new OpenAI({
        apiKey: 'your-api-key',       // 替换为您的 GogoToken API 密钥
        baseURL: 'https://gogotoken.org/v1', // GogoToken 统一端点
      });

      const completion = await client.chat.completions.create({
        model: 'claude-opus-4-8',
        messages: [{ role: 'user', content: '用一句话介绍量子计算' }],
      });

      console.log(completion.choices[0].message.content);
      ```

      ```python Python
      from openai import OpenAI

      client = OpenAI(
          api_key="your-api-key",           # 替换为您的 GogoToken API 密钥
          base_url="https://gogotoken.org/v1",  # GogoToken 统一端点
      )

      completion = client.chat.completions.create(
          model="claude-opus-4-8",
          messages=[{"role": "user", "content": "用一句话介绍量子计算"}],
      )

      print(completion.choices[0].message.content)
      ```

      ```bash cURL
      curl https://gogotoken.org/v1/chat/completions \
        -H "Content-Type: application/json" \
        -H "Authorization: Bearer your-api-key" \
        -d '{
          "model": "claude-opus-4-8",
          "messages": [{"role": "user", "content": "用一句话介绍量子计算"}]
        }'
      ```
    </CodeGroup>
  </Step>
</Steps>
