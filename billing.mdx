本站按模型类型采用不同计费方式：**Seedance 视频按输出 token 据实结算**、**Kling 视频按上游积分据实结算**、**图片按张固定价**、**素材登记按次少量计费**。实际扣费以控制台日志为准。

## Seedance 2.0 视频（Token 计费）

`doubao-seedance-2-0`、`doubao-seedance-2-0-fast`、`doubao-seedance-2-0-mini` 按输出 token 据实计费。任务提交时会进行额度校验，任务完成后按实际 `completion_tokens` 差额结算。

参考：[火山引擎 Seedance 2.0 计费文档](https://www.volcengine.com/docs/82379/1544106?lang=zh#02affcb8)

### doubao-seedance-2-0

旗舰模型，支持 1080p。

| 输出分辨率 | 纯文本 / 文生（无媒体输入） | 含媒体输入 |
| --- | --- | --- |
| 1080p | 51.00 元/百万 token | 31.00 元/百万 token |
| 480p / 720p | 46.00 元/百万 token | 28.00 元/百万 token |

### doubao-seedance-2-0-fast

快版模型，不支持 1080p。

| 输出分辨率 | 纯文本 / 文生（无媒体输入） | 含媒体输入 |
| --- | --- | --- |
| 480p / 720p | 37.00 元/百万 token | 22.00 元/百万 token |

### doubao-seedance-2-0-mini

轻量模型，不支持 1080p。

| 输出分辨率 | 纯文本 / 文生（无媒体输入） | 含媒体输入 |
| --- | --- | --- |
| 480p / 720p | 23.00 元/百万 token | 14.00 元/百万 token |

<Note>
  暂不支持离线推理。带 `-480p` 等后缀的旧别名模型仍按秒计费，见控制台定价。
</Note>

### 示例计算

`doubao-seedance-2-0-mini`，480p 不含视频，生成 108,900 tokens：

```text
参考费用 = 108900 / 1,000,000 × 23.00
        ≈ ¥2.50
```

`doubao-seedance-2-0`，720p 不含视频，生成 108,000 tokens：

```text
参考费用 = 108000 / 1,000,000 × 46.00
        = ¥4.968
```

## Kling 视频（积分计费） {#kling-视频积分计费}

可灵 Kling 模型（`kling-v2-5-turbo`、`kling-v2-6`、`kling-v3`）通过 TokenHub 按**积分**计费。模型定价与预扣上限一致时，参考零售价为 **¥1 / 积分**；若 API Key 所属分组配置了倍率，实收会乘以该倍率。任务提交时按模型**预扣上限**冻结额度，成功后按上游 `final_unit_deduction` 实收并退还差额，**失败则退还全部预扣**。

| 模型 | 预扣上限（积分） | 参考预扣金额 |
| --- | --- | --- |
| `kling-v2-5-turbo` | 5 | ¥5 |
| `kling-v2-6` | 12 | ¥12 |
| `kling-v3` | 45 | ¥45 |

### 结算示例

`kling-v2-5-turbo`，5 秒 720P 成片，上游扣 **1.5 积分**：

```text
预扣     = ¥5.00（提交时）
实收     = 1.5 × ¥1/积分 × 分组倍率（示例倍率 1.0）= ¥1.50
退款     = ¥5.00 − ¥1.50 = ¥3.50
```

使用日志「任务结算详情」可查看上游积分、视频时长与结算公式。各档位按秒单价见 [Kling 模型与计费](/kling-models)。

<Note>
  Kling **不要**使用 Seedance 的 `doubao-seedance-*` 模型名；接口路径为 `/kling/v1/videos/*`，与 `/v1/video/generations`（SD2）相互独立。
</Note>

## 素材管理（按次计费）

| 接口 | 参考单价 |
| --- | --- |
| `CreateAssetGroup` | 免费 |
| `CreateAsset` | ¥0.10 / 次 |

`CreateAsset` 用于将公网图片/视频 URL 登记到素材组，便于在图生视频中引用 `asset://`。详见 [素材管理](/assets)。

## 豆包对话模型（按 Token 计费）

`doubao-seed-2.0-pro` 等对话模型通过 `POST /v1/chat/completions` 调用，按输入与输出 token 计费。`doubao-seed-2.0-pro` 按上下文长度分档计价。详见 [对话模型](/chat-models)。

## 结算方式

- **Seedance 视频**：任务完成后根据 `completion_tokens`、模型、分辨率与是否含媒体输入计算最终费用。
- **Kling 视频**：任务完成后根据上游 `final_unit_deduction`（积分）结算，支持小数积分。
- **图片**：请求成功即扣费。
- **素材**：`CreateAsset` 成功即扣费。
- **对话**：按每次请求的 prompt / completion token 计费。

<Info>
  不同套餐、分组或活动价格可能影响最终扣费。建议以控制台定价与请求日志作为结算依据。
</Info>

## 查看用量

- 控制台日志：每次请求的扣费详情
- 控制台定价：各模型倍率与参考单价
- 控制台钱包：余额与充值

## 预扣与结算

- **Seedance 视频**：任务提交时额度校验；完成后按实际输出 token 差额结算；余额不足时返回 HTTP 402。
- **Kling 视频**：提交时按模型预扣上限冻结；成功后按上游积分实收并自动退款；详见 [Kling 模型与计费](/kling-models)。
