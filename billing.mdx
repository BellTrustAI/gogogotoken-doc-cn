`doubao-seedance-2-0` 和 `doubao-seedance-2-0-fast` 按输出 token 据实计费。任务提交时会进行额度校验，任务完成后按实际用量结算。

## 火山引擎官方定价

参考：[火山引擎 Seedance 2.0 计费文档](https://www.volcengine.com/docs/82379/1544106?lang=zh#02affcb8)

### doubao-seedance-2-0

旗舰模型，支持 1080p。

| 输出分辨率 | 输入不含视频 | 输入包含视频 |
| --- | --- | --- |
| 1080p | 51.00 元/百万 token | 31.00 元/百万 token |
| 480p / 720p | 46.00 元/百万 token | 28.00 元/百万 token |

### doubao-seedance-2-0-fast

快版模型，不支持 1080p。

| 输出分辨率 | 输入不含视频 | 输入包含视频 |
| --- | --- | --- |
| 480p / 720p | 37.00 元/百万 token | 22.00 元/百万 token |

<Note>
  暂不支持离线推理。
</Note>

## 结算方式

任务完成后，系统会根据模型服务返回的真实 `completion_tokens`、所选模型、输出分辨率和输入类型计算最终费用。实际扣费以控制台日志为准。

<Info>
  不同套餐、分组或活动价格可能影响最终扣费。建议以控制台定价与请求日志作为结算依据。
</Info>

## 示例计算

`doubao-seedance-2-0`，720p 不含视频，生成 108,000 tokens：

```text
参考费用 = 108000 / 1,000,000 * 46.00
        = ¥4.968
```

## 查看用量

- 控制台日志：每次请求的扣费详情
- 控制台定价：各模型倍率与参考单价
- 控制台钱包：余额与充值

## 预扣与结算

任务提交时会进行额度校验。任务完成后根据实际输出 token 数结算；余额不足时返回 HTTP 402。
