## 视频代理下载

代理下载已生成的视频内容。支持 API Key 或 Session 认证。

<RequestExample>
```bash cURL
curl -o output.mp4 https://gogogotoken.com/v1/videos/task_1234567890/content \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>

<ResponseExample>
```text 200
<binary mp4 content>
```
</ResponseExample>

## 路径参数

<ParamField path="task_id" type="string" required>
  视频生成任务 ID。
</ParamField>

## 请求头

<ParamField header="Authorization" type="string" required>
  API Key 鉴权信息，格式为 `Bearer YOUR_API_KEY`。
</ParamField>

<Warning>
  **Remix 混剪（`POST /v1/videos/{video_id}/remix`）不属于 Seedance 官方能力**，仅适用于 OpenAI Sora 兼容任务。Seedance 2.0 请使用 [视频生成](/video-generations) 重新提交任务。
</Warning>
