**gogogotoken.com** 国内站文档提供基于豆包 Seedance 2.0 / Seedream / Seed 对话模型的 AI 图像、视频与对话 API，并通过腾讯 TokenHub 提供**可灵 Kling 原生视频模型**（`kling-v2-5-turbo` / `v2-6` / `v3`），同时兼容 OpenAI Video、Jimeng 即梦接口格式。

<Note>
  所有接口统一使用 `https://gogogotoken.com` 作为 Base URL，并通过 Bearer Token 鉴权。
</Note>

<Note>
  本站 **Seedance 2.0（SD2）官方用法** 即本文档所述接口：`POST /v1/video/generations` 提交视频、`POST /v1/seedance/asset/*` 管理素材。平台在服务端将你提交的字段转换为火山 ARK 上游格式（如 `content[]`、`duration`、`resolution`），你无需直接调用 `/api/v3/contents/generations/tasks`。
</Note>

## 快速接入

1. 获取 API Key
2. 将 Base URL 指向 `https://gogogotoken.com`
3. 使用 `Authorization: Bearer YOUR_API_KEY` 调用接口

```http
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
```
