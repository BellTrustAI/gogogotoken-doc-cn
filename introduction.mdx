## 快速接入

只需三步，即可将 GogoToken 接入你的应用：

<Steps>
  <Step title="注册并充值">
    访问 [gogotoken.org/register](https://gogotoken.org/register) 完成注册（邀请制，按节奏开放），然后在控制台充值余额。
  </Step>
  <Step title="创建 API 密钥">
    在控制台的密钥管理页面生成你的专属 API Key，妥善保存（仅显示一次）。
  </Step>
  <Step title="修改 baseURL，立即调用">
    将你代码中的 `baseURL` 改为 `https://gogotoken.org/v1`，其余参数保持不变。
  </Step>
</Steps>

```javascript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'your-gogotoken-api-key',
  baseURL: 'https://gogotoken.org/v1', // 只需改这一行
});

const response = await client.chat.completions.create({
  model: 'claude-opus-4-8',
  messages: [{ role: 'user', content: '你好！' }],
});

console.log(response.choices[0].message.content);
```
