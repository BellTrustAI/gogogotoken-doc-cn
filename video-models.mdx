Seedance 2.0 视频生成接口提供 ARK 原生模型，按**输出 token** 据实计费，支持 480p、720p、1080p、4K（标准版；mini 仅 480p / 720p）。

<Note>
  本页仅覆盖 **Seedance（豆包）** 模型。可灵 **Kling** 视频（积分计费、`kling-v2-*` / `kling-v3`）见 [Kling 模型与计费](/kling-models)。
</Note>

<Note>
  费率对齐 **火山引擎 ARK 官方人民币价目**（2026-06）。实际扣费以任务完成后的 `completion_tokens` 为准。
</Note>

## 可用模型

| 模型 | 说明 | 最高分辨率 | 媒体输入 |
| --- | --- | --- | --- |
| `doubao-seedance-2-0` | ARK 原生旗舰模型，兼容 OpenAI Video API，任务完成后自动差额结算 | 4K | 支持 |
| `doubao-seedance-2-0-fast` | 更快、成本更低 | 720p | 支持 |
| `doubao-seedance-2-0-mini` | 轻量版，适合低成本预览与批量试片 | 720p | 支持 |

| --- | --- | --- |
| 480p / 720p | ¥46 | ¥28 |
| 1080p | ¥51 | ¥31 |
| 4K | ¥26 | ¥16 |

### `doubao-seedance-2-0-fast`

| 场景 | CNY / 百万 tokens |
| --- | --- |
| 纯文本 / 文生（无媒体输入） | ¥37 |
| 含媒体输入 | ¥22 |

### `doubao-seedance-2-0-mini`

轻量模型**不支持 1080p / 4K**，传 `1080p` 或 `4k` 会被服务端拒绝。

| 场景 | CNY / 百万 tokens |
| --- | --- |
| 纯文本 / 文生（无媒体输入） | ¥23 |

<Note>
  **媒体输入**包括：`images[]`、公网 `image_url` / `video_url` / `audio_url`，以及 `metadata.content[]` 中的 `asset://` 首帧或参考素材。不仅限于「视频 URL」字段。
</Note>

## 分辨率

可通过 `size` 或 `metadata.resolution` 指定输出分辨率。

```json
{
  "size": "720p",
  "metadata": {
    "resolution": "720p",
    "ratio": "9:16"
  }
}
```

<Note>
  分辨率须使用官方枚举 `480p` / `720p` / `1080p` / `4k`（标准版）。可通过 `metadata.resolution` 或 `size` 传入（优先级：`metadata.resolution` → `size` → 默认 `720p`）。传像素串（如 `1280x720`）会返回 `400 invalid_resolution`。画幅用 `metadata.ratio` 指定。`doubao-seedance-2-0-fast` 与 `doubao-seedance-2-0-mini` 传 `1080p` / `4k` 会被拒绝。
</Note>

## 任务模式

| 模式 | 接口 | 说明 |
| --- | --- | --- |
| 文生视频 | `POST /v1/video/generations` | 仅传提示词生成视频 |
| 图生视频 | `POST /v1/video/generations` | 传入首帧图片、参考图片或 `metadata.content[]` |
| 任务查询 | `GET /v1/video/generations/{task_id}` | 轮询查询任务状态和结果 |
