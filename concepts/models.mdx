## 查询可用模型列表

你可以通过以下 API 请求实时获取当前所有可用模型：

```bash
curl https://gogotoken.org/v1/models \
  -H "Authorization: Bearer your-api-key"
```

返回结果为标准 OpenAI 格式的模型列表，包含每个模型的 `id`、`object` 及上线时间等字段。

## 在请求中指定模型

GogoToken 与 OpenAI SDK 完全兼容，只需在初始化时将 `baseURL` 指向 GogoToken 端点，然后在请求中通过 `model` 字段指定目标模型：

```javascript
import OpenAI from "openai";

const client = new OpenAI({
  apiKey: "your-api-key",
  baseURL: "https://gogotoken.org/v1",
});

const response = await client.chat.completions.create({
  model: "claude-opus-4-8", // 在此指定模型
  messages: [
    { role: "user", content: "请用简洁的语言解释量子纠缠。" }
  ],
});

console.log(response.choices[0].message.content);
```

将 `model` 字段替换为上表中任意 **LIVE** 状态的模型 ID 即可切换模型，无需修改其他代码。

<Note>
  模型列表持续更新，建议调用 `GET /v1/models` 获取最新可用模型，以确保你的应用始终使用正确的模型 ID。
</Note>
