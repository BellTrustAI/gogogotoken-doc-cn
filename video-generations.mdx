视频生成是异步任务。提交后接口会立即返回 `task_id`，需要通过 [任务查询](/video-tasks) 轮询获取结果。

<Note>
  本文档即 **SD2 官方用法**。`prompt`、`seconds`、`images`、`metadata.*` 等字段由平台转换为 ARK 上游请求体，语义与豆包 Seedance 2.0 一致。
</Note>

<RequestExample>
```bash 文生视频
curl -X POST https://gogogotoken.com/v1/video/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedance-2-0-fast",
    "prompt": "一款智能手表在白色展示台上缓慢旋转，柔和棚拍光线，镜头平稳推进",
    "seconds": "5",
    "size": "480x854"
  }'
```

```bash 图生视频
curl -X POST https://gogogotoken.com/v1/video/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedance-2-0",
    "prompt": "基于产品首帧生成一段展示视频，镜头缓慢推进，突出产品材质和屏幕细节",
    "images": ["https://assets.example.com/product-watch.jpg"],
    "seconds": "5",
    "size": "720x1280"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": "task_1234567890",
  "task_id": "task_1234567890",
  "object": "video",
  "model": "doubao-seedance-2-0-fast",
  "status": "queued",
  "progress": 0
}
```

```json 402
{

    "code": "insufficient_quota"
  }
}
```
</ResponseExample>

## 接口说明

<Note>
  通常需要 30-120 秒生成完成。建议每 5-8 秒轮询一次。
</Note>

## 文生视频

```bash
curl -X POST https://gogogotoken.com/v1/video/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedance-2-0-fast",
    "prompt": "一款智能手表在白色展示台上缓慢旋转，柔和棚拍光线，镜头平稳推进",
    "seconds": "5",
    "size": "480x854"
  }'
```

## 图生视频

基础模式传入 `images` 作为首帧或参考图片。

```bash
curl -X POST https://gogogotoken.com/v1/video/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedance-2-0",
    "prompt": "基于产品首帧生成一段展示视频，镜头缓慢推进，突出产品材质和屏幕细节",
    "images": ["https://assets.example.com/product-watch.jpg"],
    "seconds": "5",
    "size": "720x1280"
  }'
```

高级模式通过 `metadata.content[]` 指定首帧、尾帧、参考图/视频/音频，字段与豆包官方 `content` 一致。

```bash
curl -X POST https://gogogotoken.com/v1/video/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedance-2-0",
    "prompt": "让产品在展示台上缓慢旋转，背景保持简洁，画面稳定且突出品牌质感",
    "seconds": "6",
    "size": "1080x1920",
    "metadata": {
      "content": [
        {
          "type": "image_url",
          "image_url": { "url": "https://assets.example.com/product-watch.jpg" },
          "role": "first_frame"
        },
        {
          "type": "video_url",
          "video_url": { "url": "https://assets.example.com/reference-motion.mp4" },
          "role": "reference_video"
        }
      ]
    }
  }'
```

使用 [素材管理](/assets) 登记的图片时，将 URL 换为 `asset://asset-xxx`（ID 格式以实际返回为准）：

```json
{
  "type": "image_url",
  "image_url": { "url": "asset://asset-20260627000800-l6jq6" },
  "role": "first_frame"
}
```

### `metadata.content` 中的 `role`

| role | 适用类型 | 说明 |
| --- | --- | --- |
| `first_frame` | `image_url` | 首帧图生视频 |
| `last_frame` | `image_url` | 尾帧（首尾帧生视频） |
| `reference_image` | `image_url` | 参考图片 |
| `reference_video` | `video_url` | 参考视频片段 |
| `reference_audio` | `audio_url` | 参考音频 |

<Note>
  若同时传 `images[]` 与 `metadata.content`，平台以 **`metadata.content` 为准**（`images[]` 不会保留）。
</Note>

## 字段映射（服务端自动完成）

你只需按下方参数提交；平台转换为 ARK 上游 JSON：

| 你提交的字段 | 上游 ARK 字段 | 说明 |
| --- | --- | --- |
| `prompt` | `content[]` 中 `type: "text"` | 必填 |
| `images[]` | `content[]` 中 `type: "image_url"` | 简单图生，不带 `role` |
| `metadata.content[]` | `content[]` | 多模态；含 `role` 时与官方一致 |
| `seconds` / `duration` | `duration` | `seconds` 优先 |
| `size` / `metadata.resolution` | `resolution` | `metadata.resolution` 优先；`size` 支持 `720x1280` 或 `720p` |
| `metadata.ratio` | `ratio` | 如 `16:9`、`9:16` |
| `metadata.generate_audio` | `generate_audio` | 是否生成音频 |
| `metadata.return_last_frame` | `return_last_frame` | 是否返回尾帧 |

## 参数说明

<ParamField header="Authorization" type="string" required>
  API Key 鉴权信息，格式为 `Bearer YOUR_API_KEY`。
</ParamField>

<ParamField header="Content-Type" type="string" required>
  固定为 `application/json`。
</ParamField>

<ParamField body="model" type="string" required default="doubao-seedance-2-0-fast">
  视频模型 ID，支持 `doubao-seedance-2-0` 或 `doubao-seedance-2-0-fast`。
</ParamField>

<ParamField body="prompt" type="string" required default="一款智能手表在白色展示台上缓慢旋转，柔和棚拍光线，镜头平稳推进">
  视频描述文本。
</ParamField>

<ParamField body="seconds" type="string" default="5">
  视频时长（秒），传字符串如 `"5"`。优先于顶层 `duration`。
</ParamField>

<ParamField body="duration" type="number">
  时长整数，兼容 OpenAI Video 客户端。若同时传 `seconds`，以 `seconds` 为准。
</ParamField>

<ParamField body="size" type="string" default="480x854">
  分辨率或像素尺寸，如 `1080x1920`、`720x1280`、`480x854`，或直接传 `480p` / `720p` / `1080p`。
</ParamField>

<ParamField body="images" type="string[]">
  公网图片 URL 列表，用于简单图生视频。多模态或需指定首帧/参考角色时，请使用 `metadata.content`。
</ParamField>

<ParamField body="metadata" type="object">
  SD2 扩展参数，字段名与上游 ARK 一致，放在 `metadata` 下提交：`content`、`resolution`、`ratio`、`generate_audio`、`return_last_frame`、`seed` 等。
</ParamField>

<ParamField body="metadata.resolution" type="string">
  输出分辨率 `480p` / `720p` / `1080p`，优先级高于 `size`。
</ParamField>

<ParamField body="metadata.ratio" type="string">
  宽高比，如 `16:9`、`9:16`。
</ParamField>

<ParamField body="metadata.content" type="object[]">
  多模态输入数组。每项含 `type`（`image_url` / `video_url` / `audio_url`）、对应 URL 对象，以及可选 `role`（见上文 role 表）。
</ParamField>

## 响应体

<ResponseField name="id" type="string">
  视频生成任务 ID。
</ResponseField>

<ResponseField name="task_id" type="string">
  视频生成任务 ID。部分响应会同时返回 `id` 与 `task_id`。
</ResponseField>

<ResponseField name="status" type="string">
  任务状态，例如 `queued`。
</ResponseField>

<ResponseField name="model" type="string">
  本次任务使用的模型 ID。
</ResponseField>
