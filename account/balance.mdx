GogoToken 采用预付费模式，使用前需先充值余额。每次 API 调用根据 Token 用量实时扣费，余额不设有效期，随时可用，未使用部分可申请退款。

## 充值

<Steps>
  <Step title="登录控制台">
    前往 [https://gogotoken.org](https://gogotoken.org) 并使用你的账号登录。
  </Step>
  <Step title="进入充值页面">
    在控制台导航栏中，点击「账户」或「充值」进入充值页面。
  </Step>
  <Step title="选择充值金额并完成支付">
    根据你的使用需求选择充值金额，按照页面提示选择支付方式并完成付款。
  </Step>
  <Step title="余额实时到账">
    支付成功后，余额将实时到账并显示在控制台首页，可立即用于 API 调用。
  </Step>
</Steps>

<Note>
  具体支付方式和最低充值金额以控制台页面为准，GogoToken 可能随时根据运营情况进行调整。
</Note>

## 查看余额与用量

登录控制台后，你可以在「账户」或「用量」页面查看以下信息：

- **实时余额**：当前账户可用余额，每次 API 调用后自动更新。
- **用量明细**：按时间段和模型分类展示的 Token 消耗详情，便于分析使用成本。
- **API 调用日志**：完整的请求记录，包括调用时间、使用模型、输入/输出 Token 数及费用，方便排查问题与对账。

## 低余额提醒

<Warning>
  当账户余额不足以支付本次 API 调用费用时，请求将被拒绝并返回余额不足错误。为避免因余额耗尽导致服务中断，建议根据日均用量保持一定的余额缓冲。请定期检查余额或关注控制台提醒，及时充值。
</Warning>

## 退款政策

<Note>
  账户中未使用的余额支持申请退款。如需退款，请联系 GogoToken 支持团队，提供账号信息及退款金额，团队将尽快处理。退款具体到账时间以实际支付渠道为准。
</Note>

如有更多关于计费、扣费或退款的疑问，请参阅 [常见问题](/support/faq)。
