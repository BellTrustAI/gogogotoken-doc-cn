GogoToken 兼容 OpenAI 的 Python SDK。只需设置 `base_url` 参数，即可将现有 Python 应用无缝切换到 GogoToken，无需修改任何其他业务代码。

## 安装

使用 pip 安装 OpenAI 官方 Python SDK：

```bash
pip install openai
```

建议在虚拟环境中安装，以避免依赖冲突：

```bash
python -m venv venv
source venv/bin/activate  # Windows 用户请使用 venv\Scripts\activate
pip install openai
```

## 同步调用

对于常规脚本和同步 Web 框架（如 Flask、Django），使用 `OpenAI` 同步客户端：

```python
from openai import OpenAI
import os

client = OpenAI(
    api_key=os.environ.get("GOGOTOKEN_API_KEY"),
    base_url="https://gogotoken.org/v1",
)

completion = client.chat.completions.create(
    model="claude-opus-4-8",
    messages=[
        {"role": "system", "content": "你是一个专业的代码助手"},
        {"role": "user", "content": "用 Python 写一个快速排序算法"}
    ],
    max_tokens=800,
)

print(completion.choices[0].message.content)
```

## 异步调用 (AsyncOpenAI)

对于异步框架（如 FastAPI、aiohttp）或需要并发处理多个请求的场景，使用 `AsyncOpenAI` 客户端：

```python
import asyncio
from openai import AsyncOpenAI
import os

client = AsyncOpenAI(
    api_key=os.environ.get("GOGOTOKEN_API_KEY"),
    base_url="https://gogotoken.org/v1",
)

async def main():
    completion = await client.chat.completions.create(
        model="claude-sonnet-4-6",
        messages=[{"role": "user", "content": "解释异步编程的概念"}],
    )
    print(completion.choices[0].message.content)

asyncio.run(main())
```

异步客户端还支持并发发送多个请求，大幅提升吞吐量：

```python
import asyncio
from openai import AsyncOpenAI
import os

client = AsyncOpenAI(
    api_key=os.environ.get("GOGOTOKEN_API_KEY"),
    base_url="https://gogotoken.org/v1",
)

async def ask(question: str) -> str:
    completion = await client.chat.completions.create(
        model="claude-sonnet-4-6",
        messages=[{"role": "user", "content": question}],
        max_tokens=500,
    )
    return completion.choices[0].message.content

async def main():
    questions = [
        "什么是深度学习？",
        "什么是自然语言处理？",
        "什么是强化学习？",
    ]
    # 并发发送所有请求
    answers = await asyncio.gather(*[ask(q) for q in questions])
    for q, a in zip(questions, answers):
        print(f"问：{q}\n答：{a}\n")

asyncio.run(main())
```

## 流式输出

使用 `stream` 上下文管理器可以实时逐字接收模型的输出，适合需要即时反馈的交互式场景：

```python
with client.chat.completions.stream(
    model="claude-opus-4-8",
    messages=[{"role": "user", "content": "写一个 Python 爬虫教程"}],
) as stream:
    for text in stream.text_stream:
        print(text, end="", flush=True)
```

如果你使用的是异步客户端，流式输出同样支持 `async with` 语法：

```python
async def stream_response():
    async with client.chat.completions.stream(
        model="claude-opus-4-8",
        messages=[{"role": "user", "content": "介绍一下 Python 的异步生态"}],
    ) as stream:
        async for text in stream.text_stream:
            print(text, end="", flush=True)

asyncio.run(stream_response())
```

<Tip>
  推荐将 API Key 设置为系统环境变量，而非硬编码在代码中：

  ```bash
  # macOS / Linux
  export GOGOTOKEN_API_KEY="your_api_key_here"

  # Windows (PowerShell)
  $env:GOGOTOKEN_API_KEY="your_api_key_here"
  ```

  也可以使用 `python-dotenv` 库，在项目根目录创建 `.env` 文件来管理环境变量，记得将 `.env` 加入 `.gitignore`。
</Tip>
