GogoToken 完全兼容 OpenAI API 协议。如果你已经在使用 OpenAI 的 Node.js SDK，只需将 `baseURL` 改为 GogoToken 的地址，无需修改任何业务逻辑。

## 安装 SDK

使用你偏好的包管理器安装 `openai` 官方 SDK：

<CodeGroup>
```bash npm
npm install openai
```

```bash yarn
yarn add openai
```

```bash pnpm
pnpm add openai
```
</CodeGroup>

## 基础配置

在初始化 `OpenAI` 客户端时，将 `baseURL` 设置为 GogoToken 的 API 地址，并传入你的 API Key：

```javascript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: process.env.GOGOTOKEN_API_KEY,
  baseURL: 'https://gogotoken.org/v1',
});
```

<Note>
  强烈建议将 API Key 存储在环境变量中（如 `GOGOTOKEN_API_KEY`），而不是直接硬编码在源代码里，以避免密钥泄露。你可以在项目根目录创建 `.env` 文件，并将其加入 `.gitignore`。
</Note>

## 发送对话请求

配置完成后，即可像使用 OpenAI 一样发起对话请求。以下示例调用 `claude-opus-4-8` 模型：

```javascript
const completion = await client.chat.completions.create({
  model: 'claude-opus-4-8',
  messages: [
    { role: 'system', content: '你是一个有帮助的 AI 助手' },
    { role: 'user', content: '解释一下什么是机器学习' },
  ],
  max_tokens: 1000,
  temperature: 0.7,
});

console.log(completion.choices[0].message.content);
```

## 流式输出 (Streaming)

将 `stream` 参数设置为 `true`，即可启用流式输出，实现打字机效果的逐字响应：

```javascript
const stream = await client.chat.completions.create({
  model: 'claude-sonnet-4-6',
  messages: [{ role: 'user', content: '写一首关于春天的诗' }],
  stream: true,
});

for await (const chunk of stream) {
  process.stdout.write(chunk.choices[0]?.delta?.content || '');
}
```

<Tip>
  对于高并发、高吞吐量的场景，推荐使用 `claude-sonnet-4-6` 模型——它在速度与质量之间取得了极佳的平衡，非常适合需要快速响应的生产环境。
</Tip>

## 多轮对话

通过在 `messages` 数组中保留历史消息，即可实现多轮对话。每次请求时将完整的对话历史传入：

```javascript
const conversationHistory = [
  { role: 'system', content: '你是一个有帮助的 AI 助手' },
];

async function chat(userMessage) {
  // 将用户消息添加到历史记录
  conversationHistory.push({
    role: 'user',
    content: userMessage,
  });

  const completion = await client.chat.completions.create({
    model: 'claude-opus-4-8',
    messages: conversationHistory,
    max_tokens: 1000,
  });

  const assistantMessage = completion.choices[0].message.content;

  // 将助手回复也加入历史记录，以备下一轮使用
  conversationHistory.push({
    role: 'assistant',
    content: assistantMessage,
  });

  return assistantMessage;
}

// 多轮对话示例
const reply1 = await chat('你好，我想学习 TypeScript');
console.log('助手:', reply1);

const reply2 = await chat('它和 JavaScript 有什么区别？');
console.log('助手:', reply2);

const reply3 = await chat('能给我推荐一些入门资源吗？');
console.log('助手:', reply3);
```
