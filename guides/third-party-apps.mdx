GogoToken 兼容所有支持自定义 OpenAI API 端点的第三方应用。只需在设置中填入 GogoToken 的 API 地址和你的密钥，即可在熟悉的工具中使用 Claude、GPT 等模型。

<Note>
  在所有第三方应用中，你需要填写以下两个关键信息：

  - **API Base URL**：`https://gogotoken.org/v1`
  - **API Key**：你的 GogoToken API Key（登录后在控制台的「API 密钥」页面获取）
</Note>

## ChatBox

[ChatBox](https://chatboxai.app/) 是一款支持多种 AI 后端的桌面客户端，适合日常对话使用。

<Steps>
  <Step title="打开设置">
    启动 ChatBox，点击左下角的**设置**图标（⚙️）。
  </Step>
  <Step title="选择 AI 服务商">
    在「模型」选项卡中，将 AI 服务商下拉菜单切换为 **OpenAI API**（或「自定义 API」）。
  </Step>
  <Step title="填入 GogoToken 信息">
    - 将 **API 域名** 或 **API Host** 填写为：`https://gogotoken.org/v1`
    - 将 **API Key** 填写为你的 GogoToken API Key
  </Step>
  <Step title="选择模型并保存">
    在模型列表中输入或选择你想使用的模型（如 `claude-opus-4-8`），点击**保存**即可开始对话。
  </Step>
</Steps>

## Open WebUI

[Open WebUI](https://openwebui.com/) 是一款自托管的 AI 交互界面，支持 OpenAI 兼容后端。

<Steps>
  <Step title="进入管理员设置">
    以管理员账号登录 Open WebUI，点击右上角头像，进入**管理员面板**。
  </Step>
  <Step title="打开连接配置">
    导航至 **设置 → 连接（Connections）**，找到 **OpenAI API** 部分。
  </Step>
  <Step title="配置 GogoToken 端点">
    - 将 **API URL** 修改为：`https://gogotoken.org/v1`
    - 将 **API Key** 填写为你的 GogoToken API Key
    - 点击右侧的**刷新**图标，验证连接是否成功
  </Step>
  <Step title="启用并选择模型">
    连接成功后，GogoToken 支持的模型将自动出现在模型列表中，在对话界面顶部选择所需模型即可。
  </Step>
</Steps>

## Cursor

[Cursor](https://www.cursor.com/) 是基于 VS Code 的 AI 代码编辑器，支持自定义 OpenAI 兼容 API。

<Steps>
  <Step title="打开 Cursor 设置">
    使用快捷键 `Ctrl+Shift+J`（macOS 为 `Cmd+Shift+J`）打开 Cursor 设置，或点击右上角齿轮图标。
  </Step>
  <Step title="找到模型配置">
    在设置页面中找到 **Models** 选项卡。
  </Step>
  <Step title="启用自定义 API">
    开启 **OpenAI API Key** 选项，然后勾选 **Override OpenAI Base URL**（覆盖 OpenAI 基础地址）。
  </Step>
  <Step title="填入 GogoToken 信息">
    - 在 **Base URL** 字段中填入：`https://gogotoken.org/v1`
    - 在 **API Key** 字段中填入你的 GogoToken API Key
    - 点击 **Verify**（验证）确认连接正常
  </Step>
  <Step title="添加自定义模型（可选）">
    如需使用 Claude 模型，在 **Model Names** 中点击 **Add model**，输入 `claude-opus-4-8` 或 `claude-sonnet-4-6` 并保存。
  </Step>
</Steps>

## Cherry Studio

[Cherry Studio](https://cherry-ai.com/) 是一款支持多模型管理的桌面 AI 客户端。

<Steps>
  <Step title="进入服务商设置">
    打开 Cherry Studio，点击左侧导航栏的**设置**，选择**模型服务**。
  </Step>
  <Step title="添加 OpenAI 兼容服务">
    点击**添加服务商**，在类型中选择 **OpenAI 兼容**。
  </Step>
  <Step title="填写 GogoToken 信息">
    - **API 地址**：`https://gogotoken.org/v1`
    - **API Key**：你的 GogoToken API Key
    - **服务商名称**：可自定义填写「GogoToken」
  </Step>
  <Step title="添加模型并保存">
    点击**获取模型列表**或手动添加 `claude-opus-4-8`、`claude-sonnet-4-6` 等模型，保存后即可在对话中使用。
  </Step>
</Steps>

## 通用配置规则

如果你使用的工具未在上方列出，可以按照以下通用步骤进行配置：

<Steps>
  <Step title="找到 API 设置入口">
    在应用的设置或偏好中，找到「OpenAI API Base URL」、「自定义 API 地址」、「API Endpoint」或类似的选项。
  </Step>
  <Step title="填入 GogoToken API 地址">
    将该字段的值设置为：

    ```
    https://gogotoken.org/v1
    ```
  </Step>
  <Step title="填入 API Key">
    在「API Key」或「Secret Key」字段中，填入你的 GogoToken API Key。
  </Step>
  <Step title="选择目标模型">
    在模型选择处，输入或选择你想使用的模型名称，例如：

    - `claude-opus-4-8`
    - `claude-sonnet-4-6`
    - `gpt-5-5`
    - `gpt-4o`
  </Step>
</Steps>

<Tip>
  如果你使用的工具不在上方列表中，只要它支持自定义 OpenAI 端点，配置方式通常与上述步骤完全相同。如遇到问题，可查阅该工具的官方文档中关于「自定义 OpenAI API」的章节，或联系 GogoToken 技术支持获取帮助。
</Tip>
